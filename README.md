# 网文创作工具使用说明

## 这是什么

一套完整的网文长篇小说创作工具，集成在 WorkBuddy 中。从设定搭建、大纲规划、逐章写作到伏笔追踪，覆盖几十万字长篇创作的全流程。

## 核心文件

```
~/.workbuddy/skills/web-novel-writer/
├── SKILL.md                          # 工作流引擎（AI 的行为规范）
├── references/
│   ├── story-config.md               # ★ 故事配置模板（你来填）
│   └── writing-standards.md          # 文风规范 & 格式标准
└── assets/
    └── chapter-template.md           # 章节输出模板
```

## 怎么用

### 第一步：填故事配置

打开 `story-config.md`，按照八个模块填写你的故事骨架：

| 模块         | 填什么                                 |
| ------------ | -------------------------------------- |
| 作品基础信息 | 题材、基调、平台、目标字数             |
| 世界观设定   | 世界背景、力量体系、势力分布、特殊规则 |
| 人物设定     | 主角、配角、反派的人物卡               |
| 故事主线     | 一句话梗概 → 三幕结构 → 关键剧情节点   |
| 分卷规划     | 每卷的核心事件和字数                   |
| 伏笔备忘     | 埋了哪些伏笔、什么时候回收             |
| 禁用语       | 平台敏感词、个人风格红线               |
| 参考作品     | 想接近哪种风格                         |

> 不必一次填完，先填「基础信息 + 世界观 + 主角」就能开工。

### 第二步：对话中触发创作

填好配置后，在对话中说"写第一章"或"帮我推大纲"，AI 会自动：

1. 读取你的 `story-config.md`
2. 加载 `writing-standards.md` 的风格设定
3. 按 `SKILL.md` 定义的流程生成内容

### 第三步：持续迭代

每写完一章，AI 会自动：

- 更新章节目录
- 记录新埋的伏笔
- 标注人物状态变化
- 产出"下章预告"

你不需要离开对话去任何地方手动维护——所有状态追踪都在这个工具链内自动完成。

## 四个创作阶段

| 阶段     | 做什么                 | 触发词示例         |
| -------- | ---------------------- | ------------------ |
| 设定搭建 | 填世界观、人物、主线   | "帮我设计力量体系" |
| 大纲规划 | 生成分章节大纲         | "帮我推前30章大纲" |
| 逐章写作 | 一章一章写正文         | "写第3章"          |
| 状态追踪 | 自动维护伏笔和人物状态 | 每章写完自动执行   |

## 交互原则

- **不替你拍板**：关键剧情走向你来定，AI 提供选项供选
- **接受推翻**：不满意随时改，不辩解
- **保持一致性**：严格遵循已设定的世界观和人物，发现矛盾主动提醒
- **主动追问**：设定模糊时及时追问，不自行脑补
